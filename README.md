# Paytm Clone (Frontend UI)

A static frontend UI clone of the Paytm web interface built using HTML and CSS.  
This project focuses on recreating the layout, navigation flow, and core screens of the Paytm platform for learning and frontend practice purposes.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Features ##

- Paytm-inspired clean and modern UI
- Homepage dashboard layout
- Wallet screen UI
- Recharge page interface
- Payments section layout
- User profile page design
- Organized assets for styles and images
- Static UI (no backend or real transactions)
